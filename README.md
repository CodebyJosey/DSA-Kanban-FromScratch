# DSA – Custom Kanban System (C#)

This project is part of the Data Structures and Algorithms course.

## Goal
Build a To-Do / Kanban system without using System.Collections.Generic.
All data structures are implemented from scratch.

## Implemented Data Structures
- Generic Array (made by Josey)
- Generic Linked List (made by Jur)
- Generic Hash Map (made by Ardy)
- Generic Binary Search Tree (made by Josey)

## Architecture
Clean Architecture:
- Domain
- Application
- Infrastructure
- Console UI

## Features
- Task CRUD
- Task assignment
- Task dependencies
- Kanban workflow
- JSON persistence
